# Python WSL Hello

Учебный проект для проверки настройки рабочей среды Python-разработки.

## Что сделано

- Установлен WSL Ubuntu
- В WSL работает Python3
- Настроен Git
- Проект создан и загружен на GitHub
- Создан базовый Python-скрипт

## Структура проекта

app/main.py - основной Python-файл  
examples/ - папка для скриншота результата работы  
.gitignore - список игнорируемых файлов  
requirements.txt - файл зависимостей  
README.md - описание проекта  

## Запуск

python3 app/main.py
